## Installation

Clone or download the repository:

```shell
$ git clone https://github.com/Chememers/Moleopoly
```

Install requirements:

```shell
$ cd Moleopoly
$ pip install -r requirements.txt
```

Run the game:
```shell
$ python main.py
```
